# Security Policy

## Current State of the Project

**Note:** This project is currently in a very early, "green" stage of development. 

While the current version is functional and can be used to build projects, it is still highly experimental. It has **not** yet been subjected to rigorous testing, security auditing, or comprehensive unit/integration tests. 

Use it at your own risk in production environments. We are actively working on improving stability and test coverage.

## Reporting a Vulnerability

Since the project is still in its early stages, the best way to report a vulnerability or any other bug is to open an issue directly in the repository. 

We appreciate all feedback, reports, and contributions that help make this project more robust and secure.
